## Installation

Install the Epistemic Me Python SDK using pip:

```bash
pip install epistemic-me
```

## Configuration

Initialize the SDK with your API key:

```python
import epistemic_me

epistemic_me.api_key = "your-api-key"
```

## Requirements

- Python 3.7 or higher
- pip package manager
